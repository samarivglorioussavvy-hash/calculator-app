# 🧮 Calculator

A simple and lightweight calculator built with Python.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Easy-to-use interface
* Fast and lightweight

## Installation

No installation required.

1. Extract the ZIP file.
2. Open `Calculator.exe`.
3. Start calculating!

## Source Code

The source code is included for learning and educational purposes. Feel free to explore how the calculator works.
